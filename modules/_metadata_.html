assoc

  • assoc<T>(type: AssociationType, target?: AssociationTarget<T>, options?: AssociationOptions): (Anonymous function)
    • Defined in metadata.ts:214

    A decorator for a model association. The association has a type and the model to associate to. All association properties must be defined using this decorator in order to be recognised by ModelSafe.

    Here the target is optional incase it is yet to be defined. Once it has been defined, it can be correctly reassociated with a target model using Model.assocate.

    see

    Model.associate

    Type parameters

    Parameters

      The type of association.

      The target model to associate to.

    • Optional options: AssociationOptions

      Any extra Sequelize attribute options required.
