ValidationFunction

ValidationFunction: function
  • Defined in validation.ts:137

A validation function that can be used to validate a model property and its value.

The function should return a promise that resolves when validation is successful, or rejects with a PropertyValidationError if the property validation failed.

Any errors rejected other than PropertyValidationError (i.e. if there's some unknown fatal error) will be coerced into a PropertyValidationError with its type marked as 'unknown'.
